1. To deploy the Swagger Documentation Tool
    *   Go to the directory:
    ```
    cd $GOPATH/src/github.com/nlp-dlt/network/documentation/
    ```
    *   Go to the directory:
    ```
    make start
    ```

2. To destroy the Monitoring Containers, i.e.,`docker`.
    *   Go to the directory:
    ```
    cd $GOPATH/src/github.com/nlp-dlt/network/documentation/
    ```
    *   Go to the directory:
    ```
    make destroy
    ```
